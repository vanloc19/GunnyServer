# Resource.Server Directory

⚠️ **NOTICE: Game resources support has been discontinued**

This directory is kept for folder structure reference only.

## Important Information

**Game resource files (images, sounds, assets) are NOT shared** to prevent abuse and unauthorized sale of source code.

### What is NOT included:
- ❌ Game image files
- ❌ Game sound files
- ❌ Game asset files
- ❌ Resource configuration files

### Directory Structure:
```
Resource.Server/
├── image/          # Image resources (NOT provided)
└── sound/          # Sound resources (NOT provided)
```

### Why?

These restrictions are in place to prevent misuse and unauthorized commercial distribution of the source code. This repository is intended for **educational purposes only** - to help developers understand project structure and architecture, not to provide a complete, runnable system.

**Note**: Game resources (images, sounds, assets) are sourced from China and are not included in this repository.
